# Maroon Furniture Website

## Description
This is a simple e-commerce website built using PHP and Bootstrap.

## Features
- Display products
- Add product (Create)
- Edit product (Update)
- Delete product (Delete)
- Product categories (Office Invetories & Furniture)

## How to Run
1. Import the file `maroon_furniture.sql` into your database
2. Place this project folder in `htdocs`
3. Run XAMPP (Apache & MySQL)
4. Open browser:
   http://localhost/maroon_furniture

## Tech Used
- PHP
- MySQL
- Bootstrap
